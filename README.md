# hospital-management-system
java-based hospital management system with patient prescription module


##  English
This project is a desktop application developed using Java Swing for managing hospital prescriptions.

### Overview
The system is designed for doctors to create and manage patient prescriptions.  
Doctors can enter patient details, select medications, define dosage, and include important information such as side effects and usage instructions.

### Features
- Doctor-based prescription system
- Enter patient information (name, ID, address, NHS number)
- Select medication and dosage
- Add number of tablets and reference number
- Specify possible side effects and storage advice
- Provide instructions for medication usage
- Generate and display formatted prescription output

### Screenshot
![App Screenshot](https://raw.githubusercontent.com/kkubrraa/hospital-management-system/main/Screenshots.jpeg)

### How It Works
When the application runs:
- The doctor enters patient information
- Selects medication and dosage
- Adds additional details (side effects, instructions, storage advice)
- The system generates a prescription
- The prescription is displayed on the right side of the screen

### Technologies
- Java
- Swing (GUI)
- Maven

## Türkçe
Bu proje, doktorların hasta reçetelerini oluşturması için Java Swing ile geliştirilmiş bir masaüstü uygulamasıdır.

### Genel Bakış
Sistem, doktorların hasta bilgilerini girerek ilaç seçmesini, doz belirlemesini ve reçete oluşturmasını sağlar.

### Özellikler
- Doktor odaklı reçete oluşturma sistemi
- Hasta bilgisi girme (isim, ID, adres, NHS numarası)
- İlaç ve doz seçimi
- Tablet sayısı ve referans numarası ekleme
- Yan etkiler ve saklama koşulları girme
- Kullanım talimatları ekleme
- Reçeteyi sağ tarafta çıktı olarak görüntüleme

### Nasıl Çalışır
Uygulama çalıştığında:
- Doktor hasta bilgilerini girer
- İlaç ve doz seçer
- Yan etkiler ve kullanım bilgilerini ekler
- Sistem reçeteyi oluşturur ve ekranda gösterir.
